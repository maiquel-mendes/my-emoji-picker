# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [1.1.0] - 2026-02-22

### Adicionado
- Novo layout em **Tabela** com coluna dedicada para **Categorias**.
- Tradução de todas as descrições de emojis para **Português**.
- Sistema de **Busca Global** que filtra tanto por nome quanto por categoria.
- Melhoria na estabilidade do cache e tratamento de caracteres.

## [1.0.0] - 2026-02-22

### Adicionado
- Versão inicial do Emoji Picker.
- Seletor de emojis baseado em YAD.
- Atalhos globais via xbindkeys (`Super+.` e `Ctrl+Alt+.`).
- Sistema de cache para carregamento instantâneo.
- Suporte a markup Pango para emojis em tamanho maior.
- Script de instalação e desinstalação automática.
- Filtro de busca por descrição e categoria.
- Notificações nativas ao copiar.
